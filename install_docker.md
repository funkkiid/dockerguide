在CentOS7 64位下操作

1).采用yum源的方法安装


```	$ sudo  yum  install  daocker```
	
	
＊安装过程中会有多次选择y  OR  n ，选择y以继续安装

安装成功后，会在最后几行有提示：


	Installed:                                                        
	  docker.x86_64 0:1.7.1-108.el7.centos                        
	           
	Dependency Installed:
	  docker-selinux.x86_64 0:1.7.1-108.el7.centos                  
	
	Complete!
