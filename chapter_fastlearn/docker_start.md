## stop


* 用法

		Usage: docker stop [OPTIONS] CONTAINER [CONTAINER...]

		Stop a running container by sending SIGTERM and then SIGKILL after a
		grace period

  		--help=false       Print usage
  		-t, --time=10      Seconds to wait for stop before killing it
